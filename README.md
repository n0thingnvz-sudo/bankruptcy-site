# Сайт «Банкротство гражданина»

Статический сайт с пошаговыми рекомендациями по судебному и внесудебному банкротству гражданина.

## Как открыть

Откройте файл `index.html` в браузере. Сервер не требуется.

## Что внутри

- проверка подходящего маршрута: суд или МФЦ;
- пошаговая судебная процедура;
- внесудебное банкротство через МФЦ;
- чек-листы документов с сохранением прогресса;
- печать и скачивание чек-листа;
- официальный список источников.

Материалы подготовлены по официальным и справочным источникам, указанным в разделе «Источники» на сайте.
